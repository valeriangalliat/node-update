NODE-UPDATE(8)
==============

NAME
----

node-update - Update a portable Node.js installation.

SYNOPSIS
--------

*node-update* [OPTION]...

DESCRIPTION
-----------

This command downloads the latest Node.js version for the specified architecture
in the specified directory, without overriding the whole installation.

OPTIONS
-------

*-a, --arch* 'ARCH'::
    The architecture to use (`x64' by default).

*-t, --target-dir* 'TARGET_DIR'::
    The target directory for the Node.js installation (default is current
    directory).
